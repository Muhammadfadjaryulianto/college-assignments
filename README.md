# College Assignments Repository

Repositori ini berisi kumpulan tugas praktikum selama masa perkuliahan Informatika saya. Proyek di sini mencakup berbagai bidang, mulai dari pengembangan backend hingga grafika komputer.

## Daftar Proyek

### 1. Bookstore Backend (CENA) (Express.js)
* **Folder:** `backend_cena`
* **Deskripsi:** Sistem backend untuk aplikasi toko buku mobile (Flutter).
* **Tech Stack:** Node.js, Express.js, Prisma, MySQL.

### 2. OpenGL 3D Graphics
* **Folder:** `opengl-3d-graphics`
* **Deskripsi:** Implementasi objek 3 dimensi dan konfigurasi pencahayaan menggunakan OpenGL.
* **Tech Stack:** C++, OpenGL Library.
